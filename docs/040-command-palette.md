# Command Palette

Quick access to documents and commands.

## Open

| Mac | Windows/Linux |
|-----|---------------|
| `⌘J` | `Ctrl+J` |

## Search Documents

Start typing to search across:
- **Title** (highest priority)
- **Tags** 
- **Content**

Fuzzy matching — "roadmp" finds "Roadmap".

## Commands

Type `/` to enter command mode.

| Command | Action |
|---------|--------|
| `/new` | Create new document |
| `/settings` | Open settings |
| `/fullscreen` | Enter fullscreen mode |
| `/dark` | Toggle dark/light mode |

## Keyboard Navigation

| Key | Action |
|-----|--------|
| `↑` `↓` | Navigate results |
| `Enter` | Select |
| `Esc` | Close |
